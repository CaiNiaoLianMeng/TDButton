# TDButton

===


* TD自定义按钮,封装常用样式,自定义缩放等

* 只需要导入 ``` #import "TDButton.h" ``` 即可使用
* 常用属性介绍:
    * ```buttonStyle``` : 按钮样式，请在Demo中查看
    * ```buttonCornerRadius```：按钮圆角大小
    * ```canHeightlight```：是否启用高亮状态(默认启用)
